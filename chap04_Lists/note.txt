list methods:
- index, returns the index number
    spam.index('hello')
- append, adds it to the back of the list
    spam.append('moose')
- insert, adds the value to the index specified
    spam.insert(1, 'chicken')
- remove, removes the first value from the list
    spam.remove('bat')
- sort , You can also pass True for the reverse keyword argument to have sort() sort thevalues in reverse orde
    spam.sort()
